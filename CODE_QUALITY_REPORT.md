## Рекомендации по улучшению

### 1. Логирование
```python
# Вместо:
print("Error:", e)

# Использовать:
import logging
logger = logging.getLogger(__name__)
logger.error(f"Error: {e}")
```

### 2. Обработка исключений
```python
# Вместо:
try:
    # code
except:
    # handle

# Использовать:
try:
    # code
except Exception as e:
    # handle
    logger.error(f"Exception: {e}")
```

### 3. Форматирование
- Использовать `black` или `autopep8` для форматирования
- Следовать PEP 8
- Разбивать длинные строки

### 4. Типизация
- Добавить type hints где возможно
- Использовать `mypy` для проверки типов
